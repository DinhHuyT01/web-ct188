Giới thiệu về CSS và cách kết nối CSS với HTML

1. CSS là gì?

CSS (Cascading Style Sheets) là một ngôn ngữ dùng để mô tả cách trình bày của tài liệu HTML. CSS cho phép bạn kiểm soát màu sắc, font chữ, khoảng cách, bố cục và nhiều khía cạnh khác của trang web, giúp trang web trở nên hấp dẫn và dễ sử dụng hơn.

2. Tại sao cần CSS?

Nếu chỉ dùng HTML trang web sẽ thiếu tính thẩm mỹ và khó bảo trì. CSS giúp tách biệt nội dung khỏi kiểu dáng, làm cho việc quản lý và cập nhật trang web trở nên dễ dàng hơn.

3. Cách kết nối CSS với HTML

Có ba cách để kết nối CSS với HTML:

  1. Inline CSS: Viết trực tiếp trong thẻ HTML. Nhanh nhưng khó quản lý
  2. Internal CSS: Viết trong phần đầu của trang HTML. Dùng cho các trang web nhỏ
  3. External CSS: Tạo một file CSS riêng biệt và liên kết nó với tài liệu HTML bằng thẻ <link>. Dùng cho các trang web lớn và phức tạp
