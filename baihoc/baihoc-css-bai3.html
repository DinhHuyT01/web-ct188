Màu sắc, font chữ và định dạng văn bản trong CSS

1. Màu sắc trong CSS

CSS cung cấp nhiều cách để định nghĩa màu sắc, bao gồm:

  • tên màu (color names): Ví dụ: `red`, `blue`, `green`
  • mã màu HEX (hexadecimal colors): Ví dụ: `#ff0000`, `#00ff00`, `#0000ff`
  • mã màu RGB (RGB colors): Ví dụ: `rgb(255, 0, 0)`, `rgb(0, 255, 0)`, `rgb(0, 0, 255)`
  • mã màu RGBA (RGBA colors): Giống như RGB nhưng có thêm kênh alpha để điều chỉnh độ trong suốt

2. Font chữ trong CSS

CSS cho phép bạn thay đổi font chữ của văn bản bằng cách sử dụng thuộc tính `font-family`. Bạn có thể chỉ định một font cụ thể hoặc một nhóm font dự phòng

3. Định dạng văn bản

CSS cung cấp nhiều thuộc tính để định dạng văn bản, bao gồm:

  • `font-size`: Để điều chỉnh kích thước font chữ
  • `font-weight`: Để điều chỉnh độ đậm của font chữ
  • `font-style`: Để điều chỉnh kiểu font chữ (in nghiêng, in thường...)
  • `text-decoration`: Để thêm hoặc loại bỏ đường gạch dưới, gạch ngang...

4. Nguyên tắc thiết kế văn bản

Khi thiết kế văn bản, bạn nên lưu ý đến các nguyên tắc sau:

  • Chọn font chữ phù hợp với nội dung và đối tượng người đọc
  • Đảm bảo độ tương phản giữa màu chữ và màu nền
  • Sử dụng kích thước font chữ hợp lý để đảm bảo tính dễ đọc
