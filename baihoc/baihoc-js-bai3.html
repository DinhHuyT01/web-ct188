Toán tử và biểu thức trong JavaScript

1. Toán tử là gì?

Toán tử là các ký tự hoặc biểu tượng được sử dụng để thực hiện các phép toán trên các giá trị và biến trong JavaScript.

2. Các loại toán tử trong JavaScript

JavaScript có nhiều loại toán tử khác nhau, bao gồm:

  • Toán tử số học: +, -, *, /, %
  • Toán tử gán: =, +=, -=, *=, /=
  • Toán tử so sánh: ==, ===, !=, !==, >, <, >=, <=
  • Toán tử logic: &&, ||, !
  • Toán tử chuỗi: + (nối chuỗi)

3. Biểu thức là gì?

Biểu thức là một đoạn mã JavaScript có thể được đánh giá thành một giá trị. Biểu thức có thể bao gồm các toán tử, biến, và giá trị.

4. Ví dụ về toán tử và biểu thức

Dưới đây là một số ví dụ về toán tử và biểu thức trong JavaScript:

  • Toán tử số học: 5 + 3, 10 - 2, 4 * 6, 20 / 5, 10 % 3
  • Toán tử gán: x = 5, x += 3 (tương đương x = x + 3)
  • Toán tử so sánh: 5 == '5' (true), 5 === '5' (false)
  • Toán tử logic: true && false (false), true || false (true)
  • Toán tử chuỗi: 'Hello' + ' ' + 'World' (kết quả là 'Hello World')

5. Thứ tự ưu tiên

JavaScript có một thứ tự ưu tiên nhất định khi đánh giá các biểu thức. Các toán tử có độ ưu tiên cao hơn sẽ được thực hiện trước. Bạn có thể sử dụng dấu ngoặc đơn () để thay đổi thứ tự này.
